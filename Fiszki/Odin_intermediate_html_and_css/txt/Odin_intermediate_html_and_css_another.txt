Odin_intermediate_html_and_css::19_advanced_grid_properties	Jeśli zdefiniujesz kolumny jako `1fr 2fr`, jak zostanie podzielona przestrzeń?	Druga kolumna otrzyma dwa razy więcej miejsca niż pierwsza (odpowiednio 1/3 i 2/3 dostępnej przestrzeni).	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Co zwróci funkcja CSS `min(100px, 50vw)`, jeśli okno przeglądarki ma szerokość 1000px (czyli 50vw = 500px)?	100px (ponieważ funkcja `min` zawsze wybiera **mniejszą** z podanych wartości).	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Co zwróci funkcja CSS `max(100px, 50vw)`, jeśli okno przeglądarki ma szerokość 1000px?	500px (ponieważ funkcja `max` zawsze wybiera **większą** z podanych wartości).	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Jaka funkcja CSS (specyficzna głównie dla Grida) pozwala zdefiniować zakres wielkości ścieżki od wartości minimalnej do maksymalnej?	`minmax(min, max)`	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Jaka jest składnia funkcji `clamp()`, która pozwala na płynne skalowanie elementu w określonych granicach?	`clamp(minimum, ideal, maximum)`	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Jaka jest kluczowa różnica między `auto-fit` a `auto-fill` w funkcji `repeat()`?	`auto-fit` **rozciąga** istniejące elementy, aby wypełniły cały wiersz (zwija puste ścieżki do 0), natomiast `auto-fill` **zachowuje** miejsce na potencjalne nowe kolumny (pozostawia puste przestrzenie), nawet jeśli ich nie ma.	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Jak wygląda &quot;święty graal&quot; responsywnego Grida (snippet CSS), który tworzy tyle kolumn, ile się zmieści, bez użycia Media Queries?	`grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))	
Odin_intermediate_html_and_css::19_advanced_grid_properties	Czy funkcja `clamp()` może być używana tylko w CSS Grid?	Nie, `clamp()` jest uniwersalną funkcją CSS (działa np. dla `font-size`, `width` itd.), w przeciwieństwie do `minmax()`, która działa głównie w Gridzie.	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Czy CSS Grid ma na celu zastąpienie Flexboxa?	Nie, są to narzędzia komplementarne: Grid do układów dwuwymiarowych, Flexbox do jednowymiarowych.	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Które narzędzie jest lepsze do projektowania &quot;Content-First&quot; (gdzie układ wynika z wielkości i zachowania treści)?	Flexbox (pozwala elementom elastycznie rosnąć/maleć w zależności od ich zawartości).	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Które narzędzie jest lepsze do projektowania &quot;Layout-First&quot; (gdzie najpierw definiujesz sztywną strukturę wierszy/kolumn, a potem wlewasz w nią treść)?	CSS Grid (pozwala na precyzyjne rozmieszczenie elementów w zdefiniowanych ścieżkach).	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Jeśli potrzebujesz wyrównać elementy tylko w jednej linii (wierszu LUB kolumnie), które narzędzie jest zazwyczaj bardziej intuicyjne?	Flexbox.	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Jeśli potrzebujesz precyzyjnie rozmieścić elementy w dwóch wymiarach jednocześnie (wiersze I kolumny), które narzędzie wybrać?	CSS Grid.	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Jaka jest popularna strategia łączenia Grida i Flexboxa w jednym projekcie?	Użycie Grida do ogólnego układu strony (makro-layout: nagłówek, sidebar, główna treść) oraz Flexboxa do układania elementów wewnątrz poszczególnych komórek (mikro-layout: np. nawigacja, karty). 	
Odin_intermediate_html_and_css::20_using_flexbox_and_grid	Co oznacza, że Flexbox jest jednowymiarowy (1D)?	Że zarządza układem tylko w jednej osi na raz (oś główna - wiersz lub kolumna), a zawijanie (wrap) tworzy po prostu nowe, niezależne linie flex.
