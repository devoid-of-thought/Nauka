Jakiej metody tablicowej używa się najczęściej w React do renderowania listy elementów na podstawie tablicy z danymi?;Używa się metody **`.map()`**. Np. `{animals.map(animal => <li>{animal}</li>)}`.
Jaki atrybut jest absolutnie wymagany przy każdym elemencie (lub komponencie) zwracanym bezpośrednio wewnątrz metody `.map()` podczas renderowania listy?;Atrybut **`key`** (np. `<li key={animal.id}>`). Pozwala on Reactowi jednoznacznie zidentyfikować elementy i śledzić, które z nich uległy zmianie, zostały dodane lub usunięte.
W jaki sposób możemy zrealizować proste warunkowe renderowanie (renderowanie wariantu A albo B) wewnątrz składni JSX?;Używając **operatora trójargumentowego (ternary operator)**. Np. `{isLoggedIn ? <LogoutButton /> : <LoginButton />}`.
W jaki sposób możemy zrealizować warunkowe renderowanie typu "wyświetl element, jeśli prawda, lub nie wyświetlaj niczego, jeśli fałsz" (A albo nic) wewnątrz JSX?;Używając **operatora logicznego AND (`&&`)**. Np. `{hasUnreadMessages && <NotificationIcon />}`. Jeśli warunek po lewej jest prawdziwy, wyrenderuje element po prawej. Jeśli fałszywy - zignoruje go.
Na jaki popularny "haczyk" (pitfall) w React trzeba uważać przy warunkowym renderowaniu za pomocą operatora `&&`, jeśli po lewej stronie znajduje się liczba (np. długość tablicy)?;Jeśli użyjemy samego `0` jako warunku (np. `{array.length && <List />}`), operator logiczny zwróci fałsz, ale React **wyrenderuje na ekranie cyfrę `0`**! Należy używać jawnych wyrażeń logicznych, np. `{array.length > 0 && <List />}`.
W jaki sposób można zaimplementować warunkowe renderowanie wewnątrz komponentu używając standardowych instrukcji `if/else`, skoro nie można ich pisać bezpośrednio wewnątrz tagów JSX?;Należy użyć instrukcji `if` jako tzw. **"strażników" (guard clauses) przed instrukcją `return`** JSX. Jeśli warunek jest spełniony, natychmiast wracamy z komponentu zwracając inny blok JSX.
(Przykładowy kod JSX) Co wyrenderuje ten kod, jeśli `isLoading` wynosi `false`? `return <div>{isLoading ? <Spinner /> : <Content />}</div>`;Wyrenderuje kontener `<div>` z komponentem `<Content />` w środku.
